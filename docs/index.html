🧩 Core Features
覆盖阅读全链路的功能矩阵

从书籍推荐与全文检索，到 AI 辅助阅读、段落级社交互动、多端无缝同步， 再到完善的后台管控体系——每一个环节都经过精心设计。

🤖
流式 AI 阅读助手
集成阿里云通义千问 / Dify 大语言模型。后端通过 Spring WebFlux + WebClient 异步调用， 前端基于 SSE (Server-Sent Events) 协议实现打字机效果的流式实时输出。 支持释疑解惑、内容摘要、文本续写、多轮上下文对话。
Spring WebFlux SseEmitter fetch-event-source 多轮对话

智能书架管理
个人书架系统，支持多端阅读进度自动追踪与同步。 书单分组、书架公开/私密切换、阅读完成度百分比进度条。

⭐
智能推荐 & 排行榜
首页热门轮播、热度排行榜、"猜你喜欢"个性化推荐。 推荐引擎综合用户历史阅读偏好与书籍热度智能生成推荐列表。
个性化推荐 热度排行榜 轮播推广

UGC 上传 & 审核
用户可自行上传书籍资源(TXT)。默认私有，可申请公开。 管理员审核通过后自动同步 Elasticsearch 索引，驳回可留原因。
私有/公开双状态 审核流转 ES 自动同步
📱
多端适配
桌面端基于 Vue3 + Element-Plus，移动端基于 Vue3 + Vant4。 两套独立前端项目共享同一套后端 API，代码完全隔离，互不干扰。
Element-Plus 桌面端 Vant4 移动端 共享 API 层
⚙️
沉浸式阅读体验 & 无障碍
四种阅读主题（默认/护眼/暗夜/高对比度适老模式），可调节字号与行间距。 集成 Web Speech API 整章朗读 + AI TTS 选段精准语音播报（支持多音色）。 青少年强制护眼模式——系统检测年龄＜18自动开启定时休息提醒且不可关闭。

🔌 API Endpoints
后端核心控制器接口清单

系统共计 15 个 Controller，严格遵循 RESTful 架构风格

控制器路由职能
SysBookController/sysBook/图书上传、审核、推荐、分类查询、热度排行
SysUserController/sysUser/用户登录鉴权、个人信息管理、封禁/解封
AuthController/auth/验证码发送、注册、验证码登录、密码重置
UserBookshelfController/bookshelf/书架增删、阅读进度同步、书架可见性切换
SearchController/search/Elasticsearch 全文检索代理层
DifyAiController/difyreading/AI 大模型流式中继（SSE 长连接）
AiController/ai/AI TTS 语音合成服务
SysCommentController/comment/整书级树形评论体系管理
ParagraphCommentController/paragraphComment/段落级精准批注、点赞、权限控制
FriendController/friend/好友搜索、请求/审批、列表管理
ChatController/chat/即时消息投递、已读标记、历史记录
BookShareController/bookShare/书籍分享推送 + WebSocket 通知
ParagraphShareController/paragraphShare/段落级分享（携带坐标定位）
BooklistController/booklist/书单 CRUD、分享链接生成
SysNoteController/sysNote/个人笔记管理 (AI 对话记录保存)
🚀 Quick Start
快速运行指南
1️⃣
环境准备
  • ✓ JDK 17+
  • ✓ Node.js v18+
  • ✓ MySQL 8.x
  • ✓ Redis
  • ✓ Elasticsearch 8.x
2️⃣
启动后端
# 配置 application.yml
mvn clean install
mvn spring-boot:run
3️⃣
启动前端
# 桌面端
cd reading-ui && npm i && npm run dev
# 移动端
cd reading-mobile-ui && npm i && npm run dev
